# Attach customer-controlled Repair Workers

Alphonse Node creates immutable leased Repair Tasks that customer-controlled workers claim under separate Agent Passports. A worker materializes an ephemeral workspace, invokes the customer's selected coding runtime and local credentials, and returns patches, regression artifacts, logs, and hashes; Alphonse stores no provider credential, and worker timeout or loss remains recoverable through lease fencing.
