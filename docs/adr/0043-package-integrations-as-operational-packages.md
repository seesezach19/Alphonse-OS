# Package integrations as Operational Packages

Workflow Runtime and Repair Delivery Adapters are versioned exports of existing Operational Packages alongside configuration schemas, event mappings, revision fingerprint rules, redaction defaults, reproduction extraction rules, health checks, compatibility declarations, and contract tests. This reuses Package identity, trust, compatibility, promotion, and rollback instead of creating another plugin system; V0.2 permits only first-party trusted executable adapter code.
