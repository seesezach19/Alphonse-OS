# Build a complete generic agent environment

Long-term Alphonse is a batteries-included, provider-neutral environment for building, operating, debugging, repairing, and governing Agent Workflows. It favors rich generic primitives, an opinionated lifecycle, replaceable intelligence, and pluggable edges: first-party SDKs and adapters are product features above stable contracts, while providers, vertical logic, adapters, and models never become core semantics.
