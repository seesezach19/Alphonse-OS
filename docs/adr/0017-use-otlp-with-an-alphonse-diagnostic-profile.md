# Use OTLP with an Alphonse Diagnostic Profile

The Diagnostic Plane accepts OpenTelemetry Protocol signals and offers thin SDKs, while Alphonse defines a separately versioned Diagnostic Profile for agent attribution, revision, workflow, privacy, and governance correlation. Evolving external GenAI semantic conventions remain adapters rather than Kernel semantics, and sensitive prompt, output, tool, and context payload capture is explicit and redactable.
