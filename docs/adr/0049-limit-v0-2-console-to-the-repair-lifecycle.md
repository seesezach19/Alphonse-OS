# Limit the V0.2 Console to the repair lifecycle

The V0.2 Builder Console contains only Workflow, Execution, Diagnostic Case, Repair, and Promotion views. These views expose integration health, exact revisions and timelines, explicit problem reporting, expected-versus-actual evidence, reproduction, worker and candidate status, node-level diffs, regression artifacts, Verification Receipts, human promotion, and rollback references; all deeper capability remains protocol-accessible rather than becoming premature UI.
