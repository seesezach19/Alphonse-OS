# Make the product protocol-first and agent-native

Agents are the primary protocol consumers, so every product capability exposes typed operations with exact references, preconditions, structured outcomes, idempotency, safe next actions, and resumable durable artifacts. Kernel and Diagnostic Protocol semantics remain separate while an Agent Gateway projects bounded task interfaces across them; the dashboard invokes the same operations as a human-oriented projection.
