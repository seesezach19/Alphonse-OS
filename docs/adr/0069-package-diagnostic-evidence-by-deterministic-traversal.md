# Package Diagnostic Evidence By Deterministic Traversal

Diagnostic evidence packaging starts from an exact violated evaluation and follows only typed inbound edges
allowed by a versioned Evidence Selection Policy. It includes required source roles, contradictions, unresolved
relationships, relevant gaps and conflicts, exact interpretation dependencies, and policy-permitted redacted
detail. Each selected receipt records why it entered scope, while the manifest accounts for omitted detail and
related excluded observations. Copying a logical-operation identity does not grant inclusion without a valid
projected edge. The result is an inspectable correlation path, not necessarily a causal path or exhaustive
export, and no model chooses the initial evidence set.
