# Instrument n8n with an Event Reporter subworkflow

The V0.2 n8n Package supplies an importable Alphonse Event Reporter subworkflow built from standard n8n Code, HTTP Request, Error Trigger, and subworkflow primitives. It pushes signed minimal Runtime Event Envelopes while Alphonse pulls detailed execution data only for investigation; a custom n8n node waits until the protocol and ergonomics are proven.
