# Prove V0.2 with one complete debug loop

V0.2 connects one existing TypeScript Agent Workflow to a customer-controlled Alphonse Node, ingests attributed traces, accepts an explicit failure report, creates a Diagnostic Case and deterministic Reproduction Bundle, commissions a Codex Repair Worker in an ephemeral workspace, verifies the targeted repair, and delivers a GitHub pull request with evidence. Broader evaluation, automatic promotion, Butler, ALPHONSE_DATA, managed hosting, multi-tenancy, and additional runtimes remain later product layers.
