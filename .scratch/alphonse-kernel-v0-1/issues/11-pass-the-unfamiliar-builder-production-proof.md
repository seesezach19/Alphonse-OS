# 11 - Pass The Unfamiliar-Builder Production Proof

**What to build:** An unfamiliar workflow-aware Builder uses public tools to build the inventory Package, hands it to a different runtime, safely recovers staging uncertainty, and completes one approved low-risk real correction that a Business Operator can explain.

**Blocked by:** 10 - Run The Repeatable Engineering Rehearsal.

**Status:** ready-for-agent

- [ ] Builder has not read Kernel source/internal schemas or previously built an Operational Package.
- [ ] Builder receives only public protocol, documentation, Builder Toolkit, running local environment, and source-system access.
- [ ] Workflow-specific path completes under eight active hours.
- [ ] Total human active attention remains under two hours.
- [ ] Different runtime accepts handoff without conversation history.
- [ ] Required staging timeout/recovery passes before production authority.
- [ ] User selects and approves one low-risk reversible non-AWS production target/effect.
- [ ] Business Operator explains identity, intent, versions, context, authority, effect, evidence, uncertainty, recovery, and final accountability through Butler.
- [ ] Five agent builders review proof; results record understanding, test requests, and supplied/paid workflow interest.
- [ ] Any direct SQL, authority bypass, hidden scaffold, secret copying, duplicate uncertain effect, or erased failure fails qualification.
